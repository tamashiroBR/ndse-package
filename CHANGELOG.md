# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-25

### Added
- Initial release as a standalone PHP package.
- Extraction of core engine from the legacy NDSE WebAPI.
- `LoadFlow` tool for steady-state analysis.
- `TransientAnalysis` tool for dynamic time-domain simulation.
- PSR-4 autoloading support.
- Support for generator models (Gen1, Gen2) and exciter models (Exc1, Exc2).
- Native PHP dense and sparse matrix math libraries.
